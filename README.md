# ООП Екзаменаційний проєкт — Варіант 2 (Геометричний агент)

## 📌 Опис проєкту
Цей проєкт реалізує AI-агента за допомогою Google ADK, який виконує обчислення площі та периметра геометричних фігур (прямокутник, коло, трикутник).

Проєкт демонструє основні принципи об’єктно-орієнтованого програмування:
- Абстракція
- Наслідування
- Поліморфізм
- Інкапсуляція

---

## ⚙️ Технології
- Python 3.10–3.12 (рекомендовано)
- Google ADK
- Poetry (або pipenv)

---

## 📦 Встановлення

```bash
poetry install
poetry add google-adk python-dotenv
