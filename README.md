## ⚙️ Технології
- Python 3.12+
- Google ADK
- Poetry / Pipenv
- OOP (абстракція, наслідування, поліморфізм, інкапсуляція)

---

## 🧠 Опис задачі

Проєкт реалізує AI-агента, який:
- приймає запити про геометричні фігури
- обчислює площу та периметр
- пояснює результати українською мовою

---

## 🏗️ Реалізація ООП

### 1. Абстракція
Створено абстрактний клас `Shape`, який визначає інтерфейс:

- `area()`
- `perimeter()`

---

### 2. Наслідування
Класи:
- `Rectangle`
- `Circle`
- `Triangle`

успадковують `Shape` і реалізують власні формули.

---

### 3. Поліморфізм
Методи `area()` та `perimeter()` перевизначаються в кожному класі,
що дозволяє використовувати єдиний інтерфейс для різних фігур.

---

### 4. Інкапсуляція
Клас `ShapeStorage` зберігає фігури у приватному списку:
- `__shapes`
- методи: `add()`, `total_area()`
